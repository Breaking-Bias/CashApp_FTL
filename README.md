## 🔧 Installation Instructions 🛠️

⚡ Requirements ⚡
- 🖥️ `git`
- 💻 `node.js`
- 🐍 `Python`
- 🗄️ `mySQL`

### To deploy the front-end:
```
cd frontend
npm install
npm run dev
```

### To launch that backend server:
```
cd backend
pip install -r requirements.txt
python server.py
```
Or just smash that play button! 🎮
